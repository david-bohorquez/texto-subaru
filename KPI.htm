Website Monitoring (synthetic Transactions) Type Monitor Site 24x7 Meanning Business Benefits
Availability Website This metric mesures if the site is up or down Avoiding negative impacts on the Brand reputation or business opportunity loss due to website unavailability.
Response Time Website This metric mesures the time taken to download the entire content of the web page Great User experience since the end user feels comfortable if the website responds as fast as they expect.
Downtimes Website This metric shows what exactly your website returned for the end-user when it is reporting your website as down. This option helps to Support Tech Team to isolate and resolve issues faster The business can resolve issues as soon as possible to minimize the impact when the website is having problems.
Throughput Website Metrics that shows the connection speed as well as the bandwidth usage of the server The business can mesure the speed connection to the technological resources where the application is hosted and identifying if the application needs more technical resources.
Web Page Speed Web Page Speed (Browser) It loads your web page in a real browser to analyze load time, response time, and status
Defacement Security Website defacement Metric alerts about Cyber attacks on your website that alter its structure with potentially hazardous content, graphics, malicious code insertions and so far. The business can be alert about potencial cyber atacks on the website.
Brand Reputation Security Brand Reputation metric checks websites against a regularly updated Web Risk list and notifies you if the website is affected. Web Risk list is a repository maintained by Google that comprises a list of URLs perceived to contain malicious code, i.e., malware, untrusted software, and social engineering tactics, such as phishing and other forms of deception. Reduce or avoid negative impacts on the Brand reputation due to cyber attacks.
SSL/TSL Certificate Security SSL/TLS Certificate metric periodically checks the validity of SSL/TLS certificates installed in your Web server from the cloud and triggers an alert before they expire Avoiding negative impacts on the Brand reputation or business opportunity loss due to website unavailability.
Domain Expiry Security Domain Expiry Metric watches the Domain Name and notifies you at a preset time period prior to the domain's expiry
Real User Monitor - RUM      
Apdex Score   Apdex score is a relative measurement of end users' satisfaction. It tells you whether your customer is satisfied or not, upon using your application. The business can see how is the user experience on the website evaluated as Satisfied, tolerated and frustated.
Average Response time   Displays the average of response time taken by all transactions in the application under specified time period. The business can measure how the website is running in terms of response time in a specific period.
User Sessions amount   Understand how users navigate through your website, and learn how much time they spend viewing each webpage, and in each session. The business can know how many sessions there are in a specific period of time
Pageviews   Denotes the total count of your page visits ie., the total of all your page views.  
Web Vitals   Web Vitals are modern user-centric metrics developed by Google that measure a user's experience loading a webpage
Cumulative Layout Shift: This score is used to determine the "visual stability" of a page. The lower the CLS score, the better the visual stability.
First Input Delay: This score is used to measure the time between when a user interacts with your site for the first time (i.e. when they click a link, tap a button, or press a key) and when the browser can reply to that interaction.
Largest Contentful Paint- It measures the time from when the page starts loading to when the largest text block or image element is rendered on the screen.
First Contentful Paint: It measures the time from when the page starts loading to when any part of the page's content is rendered on the screen.		 The web vitals permit visualizing the customer experience on the website, and the business can evaluate if the application is working properly or is designed as the market expects.
